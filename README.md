# Eiad Portfolio

Personal portfolio website.

## Tech
- React + TypeScript
- Vite
- Deploy: Vercel (connected to GitHub)

## Local development
```bash
npm install
npm run dev
```

## Production build
```bash
npm run build
npm run preview
```

## Quality
```bash
npm run lint
npm run format
npm run typecheck
```
